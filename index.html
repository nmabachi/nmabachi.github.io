PROJECTS

YelpCamp

Simple Yelp clone for reviewing campsites. Users can create new camps, leave commentary on existing sites and edit or delete their own submissions. Implements Node.js/Express, Mongoose/MongoDB, Passport.js and Bootstrap.

ReactReduxBlog

Single-page application written in React with Redux for application state management. Implements React, Redux, react-router, redux-form, redux-promise and axios. Backend API is currently handled by reduxblog.herokuapp.com.

ReactReduxWeather

Simple weather application written in React while using Redux for application state management. Implements React, Redux, react-sparklines, redux-promise and Bootstrap. Uses both openWeatherMap and the Google Maps API.
